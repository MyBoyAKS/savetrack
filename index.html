🔧 Firebase Setup Required for Sync
To enable cross-device sync, you need a free Firebase project:
  1. Go to console.firebase.google.com
  2. Create a new project (any name)
  3. Add a Web app, copy the config object
  4. Enable Authentication → Google
  5. Enable Firestore Database
  6. Paste your config JSON below and tap Save
